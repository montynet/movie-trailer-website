This program uses Pytho programming language to create a movie website.

Instructions to run the application include:
1. Unzip the films.zip file
1. Double click the fresh_totamotoes.html page
2. Allow blocked content for the program to run.

Trouble shooting:
If the program does not run. 
Compile the program again by opening the Pytho's IDLE and run the file myFavoriteFilms.py
